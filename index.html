RAG知识库系统

📝 项目描述：

基于 RAGFlow 的本地 AI 知识库系统，支持上传课程文档（PDF/Word/TXT），通过 RAG（检索增强生成）技术实现精准问答。可离线运行，保护隐私。


🔧 使用技术：

• Docker + RAGFlow 镜像部署

• DeepSeek / 通义千问 大模型

• 文本嵌入模型（text-embedding-v3）


📚 应用场景：

• 课程笔记快速检索

• 考试重点智能问答

• 论文摘要自动生成

AI答辩模拟助手

📝 项目描述：

AI 驱动的答辩练习系统，模拟老师与学生之间的对话场景。支持语音识别输入、AI 生成追问、语音播报回答，帮助学生在家自主练习答辩。


🔧 使用技术：

• Python + Whisper（语音识别）

• Edge TTS（中文语音合成）

• DeepSeek API（智能问答生成）

• PyQt5（桌面 GUI）


🚀 项目状态：

设计中，预计 2026 年暑假完成原型。
